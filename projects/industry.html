Industrial Lab Prototype

Two Raspberry Pi Pico 2W boards with sensors, a Raspberry Pi gateway, and a monitoring laptop.

Repository description

This repository contains the code and configuration files used to develop the prototype for my Final Degree Project. The lab setup includes two Raspberry Pi Pico 2W boards with sensors, a Raspberry Pi as a gateway, and a laptop for the monitoring/supervision side.

Architecture (high level)

  • Pico 2W nodes collect sensor data.
  • Raspberry Pi acts as gateway and central point.
  • Laptop used for monitoring and supervision.
